# Changelog VibeMode

## 1.1.25
- [ADDED] Кнопка «Подписка» теперь открывает `vibemod.pro` с авторизационным токеном пользователя.
- [ADDED] Создан лендинг `site/` для `vibemod.pro` с тремя уровнями подписки: Babby, GigaChad, Duke.
- [IMPROVED] Домен проекта обновлён с `vibemode.net` на `vibemod.pro`.

- [ADDED] Интеграция с Zed: вкладка конфигурации, автоматическая настройка OpenAI-совместимого провайдера VibeMode в `settings.json`, сброс конфигурации и запуск Zed с `VIBEMODE_API_KEY`.
- [IMPROVED] Codex: добавлен выбор модели из списка доступных на сервере вместо захардкоженного `gpt-4o`.
- [IMPROVED] Обновлены CSS-стили контент-панелей для корректной прокрутки кастомных вкладок.


- [REMOVED] OpenCode полностью убран из приложения: вкладка, Tauri-интеграция и иконки удалены.
- [IMPROVED] Codex стал главной и единственной активной вкладкой при запуске.
- [ADDED] Интеграция Codex CLI с VibeMode API: одна кнопка "Подключить" автоматически записывает `~/.codex/config.toml` и `~/.codex/env`, подставляя API-ключ из памяти приложения — ручной ввод не нужен.
- [FIXED] Иконка Codex в тёмной теме — применён корректный инверт фильтр.
