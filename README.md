## Установка

<details>
<summary><b>🍎 macOS</b></summary>

1. Скачай `.dmg` файл
2. Открой его и перетащи **VibeMode** в папку **Applications**
3. При первом запуске: правой кнопкой → **Открыть** (обход Gatekeeper)

</details>

<details>
<summary><b>🪟 Windows</b></summary>

1. Скачай `.exe` установщик
2. Запусти и следуй инструкциям
3. Если появится SmartScreen — нажми **Подробнее** → **Запустить всё равно**

</details>

<details>
<summary><b>🐧 Linux (deb)</b></summary>

```bash
sudo dpkg -i VibeMode_0.1.19_amd64.deb
# или для ARM:
sudo dpkg -i VibeMode_0.1.19_arm64.deb
```

</details>

<details>
<summary><b>🐧 Linux (rpm)</b></summary>

```bash
sudo rpm -i VibeMode-0.1.19-1.x86_64.rpm
# или для ARM:
sudo rpm -i VibeMode-0.1.19-1.aarch64.rpm
```

</details>

## Автообновление

VibeMode проверяет наличие обновлений при каждом запуске и устанавливает их автоматически (macOS). Windows и Linux — уведомление с кнопкой скачивания.
